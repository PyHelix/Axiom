### 2b. Query the DB for experiment families with recent results
```sql
SELECT
  REGEXP_REPLACE(w.name, '_[0-9a-f]{4,8}(_[0-9]+)?$', '') as family,
  SUM(r.server_state=5 AND r.outcome=1) as completed,
  COUNT(DISTINCT r.hostid) as hosts
FROM result r
JOIN workunit w ON r.workunitid=w.id
WHERE w.name LIKE 'exp_%'
  AND r.received_time >= UNIX_TIMESTAMP(CURDATE())
GROUP BY family
HAVING completed >= 10
ORDER BY completed DESC;
```

This gives you a ranked list of experiment families with results from today — without touching the filesystem. Skip any that are:
- Already CONFIRMED or REJECTED in findings_summary.txt
- In the retirement registry
- Already in the master dedup file with a Discovered date BEFORE today (see 2c below)

```bash
find /opt/axiom/experiment_results/sanitized/ -name "exp_FAMILY*_safe.json" -daystart -mtime 0 | \
  python3 -c "
import sys, json
files = [l.strip() for l in sys.stdin if l.strip()]
scored = []
for f in files:
    try:
        d = json.load(open(f))
        vals = d.get('experiment_result', {})
        if isinstance(vals, dict):
            vals = vals.get('experiment_result', [])
        if isinstance(vals, list) and len(vals) > 0:
            scored.append((float(vals[0]), f))
    except: pass
scored.sort(reverse=True)
for _, f in scored[:100]:
    print(f)
" | while read f; do cat "\$f"; echo "---"; done
```

**Why fitness sorting?** New experiments include a `_fitness` score at index 0 of their result array. For bisection experiments, fitness = 1/bracket_width (tighter bracket = higher score). The science step analyzes the deepest results first — like a chess engine trusting its deepest search. For older experiments without `_fitness`, index 0 is `trials_completed` which is a reasonable depth proxy.

Individual pulls up to 300 files per experiment family are fine since they're filtered by name.

### 3c. Report: Results and Conclusion

For each experiment you investigated, report:

```
EXPERIMENT: name (N total results available, N you sampled, N hosts)
  Discovered: YYYY-MM-DD HH:MM UTC
  Top fitness: [top 3 fitness scores from the sorted sample, e.g. "142.9 (bw=0.007, N=2048), 66.7 (bw=0.015, N=1024), 33.3 (bw=0.030, N=512)"]
  Results: [your observations from the raw data — directions, magnitudes, consistency across the highest-fitness results]
  Conclusion: STATUS. Your scientific interpretation, 2-3 sentences.
```

**CONCLUSION FORMAT IS CRITICAL — follow these rules exactly:**
- The Conclusion MUST be a single line starting with `Conclusion: STATUS.` followed by explanation text
- STATUS is one of: CONFIRMED, REJECTED, PRELIMINARY, NO EFFECT, SKIP
- Explanation text goes on the SAME LINE after the status, separated by a period and space
- NEVER use `Conclusion: : STATUS` (double colon)
- NEVER put just `Conclusion: REJECTED` with no explanation — always add 2-3 sentences
- NEVER put raw stats (d=, seeds=, fraction=) as the conclusion — those go in "Results"
- Good: `Conclusion: CONFIRMED. Ring-curvature forcing clearly suppresses defect density compared to straight stripes across all sampled results, with gentle curvature consistently outperforming tight curvature as hypothesized.`
- Bad: `Conclusion: REJECTED`
- Bad: `Conclusion: : NO EFFECT`

**Decision thresholds (guidelines, not rigid rules — use judgment):**
- Fewer than 10 results available → SKIP unless the effect is blindingly obvious
- All sampled results point the same direction with large magnitudes → likely CONFIRMED
- Mixed directions, small magnitudes, or inconsistent across samples → NO EFFECT or PRELIMINARY
- Clear effect opposite to hypothesis → REJECTED

## Step 4: Update findings_summary.txt (current-day only)

**Findings use a current-day window.** Only today's findings are kept; older entries are pruned.

### 4a. Prune old entries

Before adding new findings, remove entries not from today from findings_summary.txt.
Each entry should have a "Last updated: YYYY-MM-DD" line. If the date is not today, remove that entire entry block.

```bash
python3 -c "
import datetime, re
path = '/opt/axiom/experiment_session/findings_summary.txt'
cutoff = datetime.datetime.now().replace(hour=0, minute=0, second=0, microsecond=0)
lines = open(path).readlines()
with open('/opt/axiom/experiment_session/archive/findings_summary_' + datetime.datetime.now().strftime('%Y%m%d') + '.txt', 'w') as f:
    f.writelines(lines)
kept = []
skip = False
for line in lines:
    m = re.search(r'Last updated:\s*(\d{4}-\d{2}-\d{2})', line)
    if m:
        entry_date = datetime.datetime.strptime(m.group(1), '%Y-%m-%d')
        if entry_date < cutoff:
            skip = True
            continue
    if skip and line.strip() == '':
        skip = False
        continue
    if not skip:
        kept.append(line)
open(path, 'w').writelines(kept)
print(f'Pruned findings not from today. Kept {len(kept)} lines.')
"
```
