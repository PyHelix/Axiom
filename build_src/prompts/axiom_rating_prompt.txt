## How to rate (0-100)

Consider ALL of these factors:

**Statistical strength (0-40 points):**
- Effect size (Cohen's d): |d| > 50 = 15pts, > 10 = 12, > 5 = 9, > 2 = 6, > 1 = 3, else 0
- Sample size: 1000+ = 10pts, 500+ = 8, 200+ = 6, 100+ = 4, 50+ = 2, else 0
- Sign consistency: 100% = 10pts, 95%+ = 8, 90%+ = 6, 80%+ = 4, else 2
- Reproducibility across hosts: 10+ hosts = 5pts, 5+ = 3, else 1

**Scientific importance (0-35 points):**
- Novelty: Does this test a non-obvious hypothesis? (0-10)
- Theoretical relevance: Does it confirm/reject a named conjecture or theory? (0-10)
- Generalizability: Could this apply beyond the specific setup tested? (0-10)
- Clarity of mechanism: Is there a clear causal explanation? (0-5)

**Conclusiveness (0-25 points):**
- Clean confirmation or clean rejection with large effect = 20-25
- Partial confirmation with caveats = 10-15
- No effect with large sample (rules something out) = 10-15
- Weak/noisy result = 0-5
- Preliminary / no valid data = N/A (do not score)
