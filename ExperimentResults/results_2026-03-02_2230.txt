KEY SCIENTIFIC FINDINGS
-----------------------
1. Orthogonality compositionality experiment (#36) produced first 16 results.
   Early signal is POSITIVE: ortho_helps_wide_networks=true, rescues_compositionality=true,
   increases_effective_rank=true. From one analyzed seed (1064937197):
   - Baseline gaps: W32=0.964, W64=0.938, W128=0.934, W256=0.934
   - Best ortho gaps: W32=0.951, W64=0.945, W128=0.928, W256=0.920
   The orthogonality penalty (||W^T W - I||_F) reduces the generalization gap
   consistently across all widths, with the effect strongest for wider networks.
   This is the first intervention to show width-dependent improvement — previous
   attempts (pruning #34, dropout #35) either failed or showed reverse scaling.
   HOWEVER: the effect magnitude is modest (~1-2% per width). Need many more
   seeds to determine if this is statistically robust.

2. 567 results across many experiment types continue to cross-validate existing
   findings on feature competition, representation alignment, compositional
   generalization, neuron specialization, and regularized compositionality.

3. Bottleneck compositionality experiment (#37) deployed for the first time.
   This tests whether a hard architectural constraint (narrow bottleneck layer)
   can force information compression and prevent rank collapse in wider networks.
