KEY SCIENTIFIC FINDINGS
=========================================================
1. COMPOSITIONALITY MECHANISM INVESTIGATION — The rank collapse hypothesis
   continues to gain support. Findings #33 (effective rank ratio plummets
   0.281→0.071 from W32→W256) and #40 (disentanglement is NOT the mechanism)
   point strongly to rank collapse as the causal driver of width-compositionality
   tradeoff. Bottleneck layers (#37) rescue via information compression, not
   rank expansion (#39). This session's new experiment directly tests causality.

2. NEW EXPERIMENT: RANK REGULARIZATION COMPOSITIONALITY — Tests whether
   explicit nuclear norm regularization (encouraging higher effective rank)
   can rescue compositional generalization in wide networks. If rank collapse
   causes poor compositionality, then preventing it should fix it. Tests
   widths 32/64/128/256 × reg strengths 0/0.001/0.005/0.01/0.05.
   Script: rank_regularization_compositionality.py
   REASONING: This is the critical causal test. We have strong correlational
   evidence (rank collapse ↔ compositionality loss) but haven't yet
   demonstrated causation. If rank regularization rescues compositionality,
   the causal chain is: width → rank collapse → poor compositionality.
   If not, rank collapse may be epiphenomenal.

3. COMPOSITIONALITY CRITICAL PERIOD (Finding #41) — 18 workunits deployed
   to 10 hosts in previous session, all still in-progress (server_state=4).
   Awaiting first results. Tests whether there is a critical training epoch
   where compositionality is lost and intervention can preserve it.

4. BOTTLENECK MECHANISM (Finding #39) — Still only 1 effective seed due to
   seed extraction bug (falls back to seed=42). Redeployed with high weight
   this session to accumulate independent replications.

5. COMBINED COMPOSITIONALITY (Finding #38) — Growing confirmation (~1750+ results).
   Bottleneck+ortho combination usually beneficial (75.8%), synergy ~82%.
   Continued deployment for further validation.
